在本文中，你将了解：

- `防抖`：**重新计时**，将多次执行变为最后一次执行
- `节流`：**隔段时间执行**，将多次执行变成每隔一段时间执行
- `clearTimeout`：注意拼写，清除 timer 后，不会将 timer 置为 0
- `setInterval`： 注意拼写，循环执行
- `newDate（）`：`+new Date()`返回时间戳，`毫秒`

### 实现防抖函数（debounce）

在事件被触发 n 秒后再执行回调，如果在这 n 秒内又被触发，则重新计时

```js
// timer 保存到全局会污染全局变量环境，所以保存在闭包内
let debounce = (fn, wait) => {
  let timer = 0;

  return function(...args) {
    if(timer) clearTimeout(timer);
    timer = setTimeout(() => {
      fn.apply(this, args);
    }, wait)
  }
}

```

### 实现节流函数（throttle）

规定在一个单位时间内，只能触发一次函数。如果这个单位时间内触发多次函数，只有一次生效

```js
// 时间戳实现，更简单，不容易犯错
let throttle = (fn, wait) => {
  let lastTime = 0;
  
  return function(...args) {
    let now = +new Date();
    if(now - lastTime > wait) {
      lastTime = now;
      fn.apply(this, args);
    }
  }
}

// 测试
// setInterval(
//   throttle(() => {
//     console.log(1)
//   }, 500),
//   1
// )
```

```js
// setTimeout 实现，容易出错，
// clearTimeout(timer) 不能把 timer 归零，记得手动赋值 0，timer = 0
let throttle = (fn, wait) => {
  let timer = 0;

  return function(...args) {
    if(!timer) {
      timer = setTimeout(() => {
        clearTimeout(timer);
        timer = 0;
      }, wait);
      fn.apply(this, args);
    }
  }
}
```
