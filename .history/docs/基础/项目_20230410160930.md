### 组件按需加载

所有 `js` 模块 打包成一个 `bundle.js` 太大了，可以把某些 `js模块` 单独打包，等要用到的时候，`webpack` 自动创建 `script` 标签去请求这个 `js 文件`

`require.ensure()`

主要用于实现代码打包分离文件，require.ensure() 是 webpack 特有的

语法
```js
require.ensure(
    [], // 依赖
    () => {// 回调函数，在这里按需引入模块和处理逻辑
        require('@/components/home')
    }, 
    'demo') // webpackChunk 名，同名的打包进同个 bundle
```
